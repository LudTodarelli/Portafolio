Portafolio Personal – Frontend

Este repositorio contiene mi portafolio personal. El objetivo del sitio es presentar mi perfil profesional, mis conocimientos técnicos y algunos proyectos realizados, utilizando tecnologías web estándar.

Cuenta con las siguientes secciones principales:
- Hola: presentación personal.
- Conocimientos: tecnologías y herramientas que manejo.
- Proyectos: muestras de trabajos realizados.
-Contacto: formulario y enlaces a redes profesionales.

La navegación es clara e intuitiva, permitiendo acceder fácilmente a cada sección desde el menú principal.

Herramientas y tecnologías utilizadas

-HTML5 
  Estructura semántica del sitio y organización del contenido.

-CSS3
  Estilos, diseño visual, layout con Flexbox y Grid, efectos hover y responsive design mediante media queries.

- JavaScript 
  Interactividad básica, como el funcionamiento del menú hamburguesa en dispositivos móviles.

- Google Fonts
  Uso de la tipografía Oswald para el diseño del sitio.

- Bootstrap Icons
  Íconos utilizados en navegación y redes sociales.

- Git y GitHub  
  Control de versiones y publicación del repositorio.

Funcionalidades

- Al clickear en los links de navegación, el sitio redirige correctamente a la sección correspondiente mediante anclas HTML.
- Al clickear en los links de contacto, se accede a las páginas externas correspondientes (LinkedIn, Behance, email).
- Diseño totalmente responsivo, adaptado a desktop, tablet y mobile.
- Menú hamburguesa funcional en resoluciones pequeñas.
- Modales para visualizar proyectos en mayor detalle.
